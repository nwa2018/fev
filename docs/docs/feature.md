# 特性

`fev`开箱即用，内置封装大量的转换器，工程化特性，只需要安装一次在全局，即可多次使用
### 转换器

- 支持`stylus`, `less`, `scss`, `sass`, `postcss`, `css module`

- 支持`typescript`, `flow`, `jsx`, `es6`

- 支持`html`, `pug`, `artemplate`

### 框架

支持`vue`, `react`

### 其他

- 支持开发`npm`组件库

- 支持`eslint`

- 支持`热重载`，`代码分割`，...

- 更多内容请阅读 [指南](docs/guide.md)
