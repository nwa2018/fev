# 开始

## 安装

!> 需要`node`版本 **8.x+**

由于依赖比较多，安装过程较慢，请耐心等待
```
npm i @fev/fev -g
```
如果安装速度很慢，可以尝试换源
```
npm i @fev/fev -g --registry https://registry.npm.taobao.org
```
## 快速开始
运行以下命令快速生成模板

> fev i

## 其他命令

- 开发 `fev d` / `fev development`

- 构建 `fev b` / `fev build`

- 查看版本号 `fev -v` / `fev --version`

- 查看所有命令 `fev` / `fev -h` / `fev --help`

## 卸载
```
npm uninstall @fev/fev -g
```
